|mac1      |MAC地址 |  操作设备MAC地址编码加密，原字段如“38：XX:XX:XX:XX:92”  |

|ip2     | ip地址 |  操作电脑IP地址编码加密  |

|geo_code     | 地理信息 |  经纬度GeoHash编码  |
